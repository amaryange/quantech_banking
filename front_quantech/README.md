## Installation et lancement local

1. Cloner le dépôt

   ```bash
   git clone https://github.com/votre-utilisateur/votre-projet.git
   ```
2. Installer les dépendances

   ```bash
   npm i
   bun i
   ```
3. Executer

   ```bash
   npm run dev
   pnpm dev
   ```
